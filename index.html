VS Code로 첫 웹사이트 만들기

이 템플릿은 하나의 HTML 파일 안에 스타일(CSS)과 스크립트(JS)를 함께 담았습니다. 초보자가 index.html 파일 하나로 구조를 익히고, 나중에 분리해 확장할 수 있도록 구성했습니다.

바로 살펴보기 문의하기

기능

반응형 레이아웃

그리드 레이아웃으로 모바일 → 데스크톱까지 자동 적응합니다.

다크/라이트 모드

prefers-color-scheme를 사용해 시스템 테마를 따라갑니다.

접근성 기본값

시맨틱 태그, aria-* 속성, 포커스 가능한 버튼 등 기본을 포함합니다.

빠른 수정 방법

텍스트를 바꿔보세요: 상단의 h1 제목이나 카드의 문장을 직접 수정하면 즉시 반영됩니다.

  1. index.html을 VS Code에서 열기
  2. Live Server로 실행(없으면 아래 안내 참고)
  3. 제목/문단/링크 내용을 바꾸고 저장(Ctrl/Cmd + S)
  4. 브라우저가 자동 새로고침되며 결과 확인
